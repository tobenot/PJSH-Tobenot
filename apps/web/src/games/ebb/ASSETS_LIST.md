# 素材需求列表 - Project: EBB (沉没都市)

## 1. 瓦片地图素材 (Tilemap Assets) - 32x32px
建议风格：写实黑暗线条风格 (Dark Realistic Line Art)，偏冷色调。

### 地形 (Terrain)
- **Ground (地面)**: 混凝土、破碎的沥青、带有裂缝的地面。
- **Water_Shallow (浅水)**: 半透明的黑色/深蓝色水面，覆盖在地面上，隐约可见底部。
- **Water_Deep (深水/深渊)**: 纯黑色或极深蓝色，无法看清底部，带有危险感。
- **Wall (墙壁)**:
  - 砖墙 (Brick Wall)
  - 混凝土墙 (Concrete Wall)
  - 带有涂鸦的废墟墙
- **Floor_Safe (安全区地面)**: 灯塔内部的木地板或金属网格地板。

### 物件/掩体 (Objects & Cover)
- **Hard Cover (硬掩体)**:
  - 混凝土柱子 (Pillar)
  - 废弃的生锈汽车 (Rusted Car)
  - 保险柜 (Safe)
  - 铁桶 (Iron Barrel)
- **Soft Cover (软掩体/可破坏)**:
  - 木箱 (Wooden Crate)
  - 办公桌 (Office Desk)
  - 玻璃窗 (Glass Window) - 破碎状态/完整状态
  - 尸体堆 (Pile of Corpses)
  - 货架 (Shelf)

## 2. 角色素材 (Character Sprites)
俯视角 (Top-down) 或 伪3D视角。

### 玩家 (Player)
- **Idle**: 站立，持有武器。
- **Move**: 行走/涉水（涉水时下半身有波纹遮挡）。
- **Attack**: 射击、挥砍。
- **Dead**: 倒在水中。

### 敌人 (Enemies)
1.  **淤泥行者 (Mud Walker)**:
    - 外观：被黑色油污包裹的僵尸，动作扭曲。
    - 动作：缓慢爬行、扑击。
2.  **模仿者 (Mimic)**:
    - 外观：穿着破烂军装或人类衣服，但肢体比例不对劲。
    - 动作：持枪射击、躲避（蹲在掩体后）。
3.  **灯塔守望者/准将 (The Commodore - BOSS)**:
    - 外观：深蓝色长风衣，脸部是黑色虚空或潜水头盔。
    - 动作：举起信号枪、挥手、拔出指挥刀。
4.  **深潜卫队 (Deep Dive Guard)**:
    - 黑色蛙人装备。

### NPC
- **走私客 (The Smuggler)**: 坐在阴影中，周围堆满杂物。

## 3. UI 图标与界面 (UI Assets)

### 核心 HUD
- **HP Icon**: 红十字或心跳图。
- **Sanity Icon**: 大脑或眼睛（代表理智）。
- **Weight Icon**: 背包或秤砣。
- **Tide Meter (潮汐表)**:
  - 进度条背景。
  - 指针/水位线。
  - 阶段标记 (低潮、涨潮、满潮)。

### 物品图标 (Item Icons)
- **Consumables**: 罐头 (Canned Food)、滤芯 (Filter)、急救包 (Medkit)、威士忌 (Whiskey)。
- **Weapons**: 旧式手枪 (Old Pistol)、消防斧 (Fire Axe)、信号枪 (Flare Gun)。
- **Quest Items**: 气象数据黑盒 (Black Box / SSD)、门禁卡 (Keycard)。

## 4. 特效素材 (VFX)
- **Danger Line (危险红线)**: 红色激光线条，带发光效果。
- **Muzzle Flash (枪口火焰)**: 瞬间闪光。
- **Impact**: 子弹击中墙壁的火花/尘土。
- **Water Ripple (水波纹)**: 角色在水中移动时的白色波纹。
- **Fire (火焰)**: 燃烧动画。
- **Smoke (烟雾)**: 阻挡视线的灰色烟雾团。
- **Signal Flare (信号弹)**: 红色、黑色光柱。

## 5. 音效需求 (Audio) - 可选
- **BGM**: 风声、海浪声、低沉的压抑旋律。
- **SFX**: 枪声、换弹声、脚步声（水声/地砖声）、怪物嘶吼、潮水上涨的警报声。
